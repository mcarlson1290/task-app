# Replit Instructions: Add Frequency Type Badges

## Goal: Add colored badges that show "WEEKLY", "BI-WEEKLY", "MONTHLY", "QUARTERLY"

## Step 1: Find Where Badges Are Currently Displayed

Look for where you see the blue RECURRING badge. The code should look something like:

```javascript
{task.recurringTaskId && (
  <span className="px-2 py-1 rounded text-xs font-bold uppercase bg-blue-600 text-white">
    RECURRING
  </span>
)}
```

## Step 2: Add Frequency Type Badge After RECURRING

Right after the RECURRING badge code, add this frequency badge:

```javascript
{/* Existing RECURRING badge */}
{task.recurringTaskId && (
  <span className="px-2 py-1 rounded text-xs font-bold uppercase bg-blue-600 text-white">
    RECURRING
  </span>
)}

{/* ADD THIS: Frequency Type Badge */}
{task.recurringTaskId && (
  <span className={`px-2 py-1 rounded text-xs font-bold uppercase text-white ml-1
    ${task.frequency === 'daily' ? 'bg-green-600' : 
      task.frequency === 'biweekly' ? 'bg-purple-600' : 
      task.frequency === 'monthly' ? 'bg-orange-600' : 
      task.frequency === 'quarterly' ? 'bg-pink-600' : 
      'bg-gray-600'}
  `}>
    {task.frequency === 'daily' ? 'WEEKLY' :
     task.frequency === 'biweekly' ? 'BI-WEEKLY' :
     task.frequency === 'monthly' ? 'MONTHLY' :
     task.frequency === 'quarterly' ? 'QUARTERLY' : 
     'CUSTOM'}
  </span>
)}
```

## Step 3: If Frequency is Missing, Try This

If the badges still don't show, the `frequency` field might be missing. Try this version that gets it from the recurring task:

```javascript
{/* Get frequency from recurringTaskId if not on task */}
{task.recurringTaskId && (
  <span className={`px-2 py-1 rounded text-xs font-bold uppercase text-white ml-1
    ${getTaskFrequency(task) === 'daily' ? 'bg-green-600' : 
      getTaskFrequency(task) === 'biweekly' ? 'bg-purple-600' : 
      getTaskFrequency(task) === 'monthly' ? 'bg-orange-600' : 
      getTaskFrequency(task) === 'quarterly' ? 'bg-pink-600' : 
      'bg-gray-600'}
  `}>
    {getTaskFrequency(task) === 'daily' ? 'WEEKLY' :
     getTaskFrequency(task) === 'biweekly' ? 'BI-WEEKLY' :
     getTaskFrequency(task) === 'monthly' ? 'MONTHLY' :
     getTaskFrequency(task) === 'quarterly' ? 'QUARTERLY' : 
     'CUSTOM'}
  </span>
)}

{/* Add this helper function */}
const getTaskFrequency = (task) => {
  // First check if task has frequency
  if (task.frequency) return task.frequency;
  
  // Fallback: try to determine from title or other fields
  // You may need to look up the recurring task by ID
  return 'daily'; // default
};
```

## Step 4: Complete Badge Set Example

Here's what the full badge section should look like:

```javascript
<div className="flex flex-wrap items-center gap-1 mb-2">
  {/* RECURRING Badge */}
  {task.recurringTaskId && (
    <span className="px-2 py-1 rounded text-xs font-bold uppercase bg-blue-600 text-white">
      RECURRING
    </span>
  )}
  
  {/* Frequency Type Badge */}
  {task.recurringTaskId && (
    <span className={`px-2 py-1 rounded text-xs font-bold uppercase text-white
      ${task.frequency === 'daily' ? 'bg-green-600' : 
        task.frequency === 'biweekly' ? 'bg-purple-600' : 
        task.frequency === 'monthly' ? 'bg-orange-600' : 
        task.frequency === 'quarterly' ? 'bg-pink-600' : 
        'bg-gray-600'}
    `}>
      {task.frequency === 'daily' ? 'WEEKLY' :
       task.frequency === 'biweekly' ? 'BI-WEEKLY' :
       task.frequency === 'monthly' ? 'MONTHLY' :
       task.frequency === 'quarterly' ? 'QUARTERLY' : 
       'CUSTOM'}
    </span>
  )}
  
  {/* Day Badge */}
  <span className="px-2 py-1 rounded text-xs font-bold uppercase bg-teal-500 text-white">
    {getDayAbbreviation(task.dueDate)}
  </span>
  
  {/* Priority Badge */}
  {task.priority && (
    <span className={`px-2 py-1 rounded text-xs font-bold uppercase text-white
      ${task.priority === 'high' ? 'bg-red-500' : 
        task.priority === 'medium' ? 'bg-yellow-600' : 
        'bg-green-500'}
    `}>
      {task.priority}
    </span>
  )}
</div>
```

## Step 5: Debug if Still Not Working

Add this console log to see what data the task has:

```javascript
{task.recurringTaskId && console.log('Task frequency:', task.title, task.frequency)}
```

## Color Scheme for Frequency Badges

- **WEEKLY** (daily): Green (`bg-green-600`)
- **BI-WEEKLY**: Purple (`bg-purple-600`)  
- **MONTHLY**: Orange (`bg-orange-600`)
- **QUARTERLY**: Pink (`bg-pink-600`)

## Expected Result

Each recurring task should show:
1. Blue **RECURRING** badge
2. Colored frequency badge (**WEEKLY**, **BI-WEEKLY**, etc.)
3. Teal day badge (**WED**)
4. Yellow/Red/Green priority badge (**MEDIUM**)

All in a nice row that wraps if needed!
