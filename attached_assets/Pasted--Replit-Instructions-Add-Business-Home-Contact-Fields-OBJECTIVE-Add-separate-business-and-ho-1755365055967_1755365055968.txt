## Step 6: Update Authentication Check

Update the login authentication to check the personal email:

```javascript
// In your authentication/permission check:
const checkUserPermissions = (loginEmail) => {
  const staffMember = staff.find(s => 
    s.personalEmail.toLowerCase() === loginEmail.toLowerCase() && 
    s.status === 'Active'
  );
  
  if (staffMember) {
    return {
      hasAccess: true,
      role: staffMember.role,
      location: staffMember.location
    };
  }
  
  return { hasAccess: false };
};
```
