# Fix Filter Bar Element Heights

## Problem
The filter bar has inconsistent heights across different elements:
- Dropdowns (Category, Status, Priority) have one height
- Search input has a different height  
- Buttons (Today, Clear Filters, New Task, Refresh Tasks) have another height

## Solution for Replit

### Set Consistent Heights for All Filter Elements

```css
/* Create a consistent height variable */
:root {
  --filter-height: 38px; /* Adjust this value as needed */
}

/* Apply to all dropdowns */
.filter-container select {
  height: var(--filter-height);
  padding: 0 12px;
  border: 1px solid #ddd;
  border-radius: 4px;
  font-size: 14px;
  background-color: white;
  cursor: pointer;
}

/* Apply to search input */
.filter-container input[type="search"],
.filter-container input[type="text"] {
  height: var(--filter-height);
  padding: 0 12px;
  border: 1px solid #ddd;
  border-radius: 4px;
  font-size: 14px;
}

/* Apply to all buttons in filter bar */
.filter-container button {
  height: var(--filter-height);
  padding: 0 16px;
  border: none;
  border-radius: 4px;
  font-size: 14px;
  cursor: pointer;
  display: inline-flex;
  align-items: center;
  justify-content: center;
  white-space: nowrap;
}

/* Date picker should also match */
.filter-container input[type="date"] {
  height: var(--filter-height);
  padding: 0 12px;
  border: 1px solid #ddd;
  border-radius: 4px;
  font-size: 14px;
}

/* Ensure vertical alignment */
.filter-container {
  display: flex;
  align-items: center; /* This ensures everything aligns vertically */
  gap: 8px;
}

/* Remove any default styling that might affect height */
.filter-container * {
  box-sizing: border-box;
  margin: 0;
}
```

### If Using Styled Components or CSS-in-JS

```typescript
const filterElementStyles = {
  height: '38px',
  padding: '0 12px',
  border: '1px solid #ddd',
  borderRadius: '4px',
  fontSize: '14px',
  boxSizing: 'border-box' as const,
  margin: 0
};

const buttonStyles = {
  ...filterElementStyles,
  padding: '0 16px',
  border: 'none',
  cursor: 'pointer',
  display: 'inline-flex',
  alignItems: 'center',
  justifyContent: 'center'
};

// Apply to components
<select style={filterElementStyles}>...</select>
<input type="search" style={filterElementStyles} />
<button style={buttonStyles}>Today</button>
```
