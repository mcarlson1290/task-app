
## Step 2: Clear All Tasks

In your task service or wherever tasks are stored:

```tsx
// Clear all existing tasks
const clearAllTasks = () => {
  // If using localStorage
  localStorage.removeItem('tasks');
  localStorage.removeItem('taskHistory');
  localStorage.removeItem('completedTasks');
  
  // If using a service
  taskService.clearAll();
  
  // Reset to empty array
  setTasks([]);
};

// Add a temporary button or run this once
clearAllTasks();
```

## Step 6: Create a Reset Function (Optional)

Add a hidden admin function to reset all data:

```tsx
// Add to a admin/debug component
const AdminReset = () => {
  const resetAllData = () => {
    if (confirm('Are you sure you want to clear ALL data? This cannot be undone.')) {
      // Clear tasks
      localStorage.removeItem('tasks');
      localStorage.removeItem('recurringTasks');
      
      // Clear education
      localStorage.removeItem('courses');
      localStorage.removeItem('courseProgress');
      
      // Clear inventory
      localStorage.removeItem('inventory');
      
      // Clear staff (but keep current user)
      const currentUser = getCurrentUser();
      localStorage.clear();
      setCurrentUser(currentUser);
      
      alert('All data cleared. Please refresh the page.');
      window.location.reload();
    }
  };
  
  // Only show for Corporate Manager
  const user = getCurrentUser();
  if (user.role !== 'Corporate') return null;
  
  return (
    <button onClick={resetAllData} style={{
      position: 'fixed',
      bottom: 20,
      right: 20,
      opacity: 0.3
    }}>
      Reset All Data
    </button>
  );
};
```

## Step 7: Initialize with Clean State

Update your initialization code to start fresh:

```tsx
// In your main App or service initialization
const initializeApp = () => {
  // Check if this is first run after cleanup
  const isCleanInstall = !localStorage.getItem('appInitialized');
  
  if (isCleanInstall) {
    // Set up clean initial state
    localStorage.setItem('appInitialized', 'true');
    localStorage.setItem('tasks', JSON.stringify([]));
    localStorage.setItem('inventory', JSON.stringify([]));
    localStorage.setItem('courses', JSON.stringify([]));
    localStorage.setItem('recurringTasks', JSON.stringify([]));
  }
};
```

## Quick One-Time Cleanup Script

Run this ONCE in your browser console while on the app:

```javascript
// Clear all localStorage data
localStorage.clear();

// Refresh the page
location.reload();
```

This will wipe everything clean for a fresh start.
