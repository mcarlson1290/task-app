# Tasks Page Filter Bar - Compact Width Fix

## The Problem
The filter elements are too wide, making it impossible to see all filters even with horizontal scrolling enabled.

## Fix: Make All Filter Elements Compact

Add this CSS to make the filters much narrower:

```css
/* Compact filter styles */
.task-filters-container {
  display: flex;
  align-items: center;
  gap: 8px; /* Reduced gap */
  padding: 12px 16px;
  overflow-x: auto;
  overflow-y: hidden;
  white-space: nowrap;
}

/* Make ALL filter elements compact */
.filter-dropdown {
  height: 32px; /* Slightly shorter */
  padding: 0 8px; /* Less padding */
  min-width: 80px; /* Much narrower */
  max-width: 120px; /* Prevent them from being huge */
  font-size: 13px; /* Slightly smaller text */
  border: 1px solid #ddd;
  border-radius: 4px;
  background-color: #f5f5f5;
}

/* Compact search bar */
.search-bar {
  height: 32px;
  width: 140px; /* Fixed narrow width */
  padding: 0 8px;
  font-size: 13px;
  border: 1px solid #ddd;
  border-radius: 4px;
}

/* Compact buttons */
.filter-button,
.add-task-btn {
  height: 32px;
  padding: 0 12px; /* Less padding */
  font-size: 13px;
  white-space: nowrap;
  border-radius: 4px;
  cursor: pointer;
}

.add-task-btn {
  background-color: #28a745;
  color: white;
  border: none;
  min-width: auto; /* Only as wide as needed */
}

.filter-button {
  border: 1px solid #ddd;
  background-color: #f5f5f5;
}

/* Compact date picker */
.date-picker {
  height: 32px;
  padding: 0 6px;
  font-size: 13px;
  border: 1px solid #ddd;
  border-radius: 4px;
  width: 100px; /* Fixed narrow width */
}

/* Even more compact on very small screens */
@media (max-width: 480px) {
  .filter-dropdown {
    min-width: 70px;
    max-width: 100px;
    font-size: 12px;
  }
  
  .search-bar {
    width: 100px;
  }
  
  .filter-button,
  .add-task-btn {
    padding: 0 8px;
    font-size: 12px;
  }
}
```

## If Still Too Wide - Ultra Compact Version

If you need them even smaller:

```css
/* ULTRA COMPACT VERSION */
.filter-dropdown {
  height: 28px;
  padding: 0 6px;
  min-width: 60px;
  max-width: 90px;
  font-size: 12px;
}

.search-bar {
  height: 28px;
  width: 100px;
  font-size: 12px;
}

.filter-button,
.add-task-btn {
  height: 28px;
  padding: 0 8px;
  font-size: 12px;
}

.date-picker {
  height: 28px;
  width: 85px;
  font-size: 12px;
}
```

## Alternative: Icon-Only Filters

For maximum space saving, use icons with tooltips:

```css
/* Icon-only compact filters */
.icon-filter {
  width: 32px;
  height: 32px;
  padding: 0;
  display: flex;
  align-items: center;
  justify-content: center;
  border: 1px solid #ddd;
  border-radius: 4px;
  background-color: #f5f5f5;
  cursor: pointer;
  position: relative;
}

/* Tooltip on hover */
.icon-filter:hover::after {
  content: attr(data-tooltip);
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  padding: 4px 8px;
  background: #333;
  color: white;
  font-size: 12px;
  border-radius: 4px;
  white-space: nowrap;
  margin-bottom: 4px;
}
```

Then use like:
```html
<button class="icon-filter" data-tooltip="Status Filter">
  <svg><!-- status icon --></svg>
</button>
```

This should make all the filters visible in your current viewport!
