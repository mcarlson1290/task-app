## 2. FILTER TASKS TO TODAY ONLY

Add a filter to only include today's tasks. After the `if (selectedDate !== today) return null;` line, add:

```javascript
// Filter tasks to only today's tasks
const todayTasks = tasks.filter(task => {
  // Completed/skipped tasks: check if completed today
  if (task.status === 'completed' || task.status === 'skipped') {
    const actionDate = (task.completedAt || task.skippedAt || task.dueDate).split('T')[0];
    return actionDate === today;
  }
  
  // Overdue tasks: show on today
  if (task.dueDate < today && (task.status === 'pending' || task.status === 'in_progress')) {
    return true;
  }
  
  // Regular tasks: due today
  return task.dueDate === today;
});
```

Then update all references from `tasks` to `todayTasks`:
- Change `tasks.filter(...)` to `todayTasks.filter(...)`
- Change the myTasks filter to use `todayTasks` instead of `tasks`
