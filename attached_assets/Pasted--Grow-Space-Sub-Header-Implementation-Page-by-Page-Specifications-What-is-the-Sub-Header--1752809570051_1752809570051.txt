# Grow Space Sub-Header Implementation - Page-by-Page Specifications

## What is the Sub-Header?

The sub-header is a **SECOND navigation/control bar** that appears BELOW your main navigation header. It contains:
- Page-specific action buttons (like "Add Item", "Create Course")
- Filter controls (dropdowns, search bars)
- Tab navigation for sub-pages
- Export/report buttons

## Step 1: Create the Sub-Header Component Files

### 1.1 Create SubHeader Component
**Create NEW file: `src/components/SubHeader.jsx`**

```jsx
import React from 'react';
import './SubHeader.css';

const SubHeader = ({ children, className = '' }) => {
  return (
    <div className={`sub-header-wrapper ${className}`}>
      <div className="sub-header-container">
        <div className="sub-header-content">
          {children}
        </div>
      </div>
    </div>
  );
};

export default SubHeader;
```

## Step 4: Connect Functionality

For each filter/button in the sub-header, connect it to your existing state/functions:

```jsx
// Example for inventory page filters
const [searchTerm, setSearchTerm] = useState('');
const [categoryFilter, setCategoryFilter] = useState('all');
const [sortBy, setSortBy] = useState('name');
const [lowStockOnly, setLowStockOnly] = useState(false);

// Filter your data based on these states
const filteredInventory = inventory
  .filter(item => item.name.toLowerCase().includes(searchTerm.toLowerCase()))
  .filter(item => categoryFilter === 'all' || item.category === categoryFilter)
  .filter(item => !lowStockOnly || item.quantity < item.minQuantity)
  .sort((a, b) => {
    if (sortBy === 'name') return a.name.localeCompare(b.name);
    if (sortBy === 'date') return new Date(b.date) - new Date(a.date);
    if (sortBy === 'value') return b.value - a.value;
  });
```

## Summary of What Goes in Each Sub-Header

1. **Tasks**: View toggle (All Tasks / Category)
2. **Inventory**: Add button + Search + Category filter + Sort
3. **Education**: Course filter tabs + Create Course button
4. **Recurring Tasks**: Just Add Recurring Task button
5. **Task Data**: View tabs + Time filter + Export button
6. **Staff Data**: View tabs + Sort + Export + Add Staff button
7. **Production Data**: View tabs (5 tabs) + Export button

Remember: The sub-header is a SECOND bar, not a replacement for your main navigation!
