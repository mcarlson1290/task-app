## Step 1: Create New Component Files (DO NOT MODIFY EXISTING FILES YET)

### 1.1 Create SubHeader Component
**Create NEW file: `src/components/SubHeader.jsx`**

```jsx
import React from 'react';
import './SubHeader.css';

const SubHeader = ({ children, className = '' }) => {
  return (
    <div className={`sub-header-wrapper ${className}`}>
      <div className="sub-header-container">
        <div className="sub-header-content">
          {children}
        </div>
      </div>
    </div>
  );
};

export default SubHeader;
```
