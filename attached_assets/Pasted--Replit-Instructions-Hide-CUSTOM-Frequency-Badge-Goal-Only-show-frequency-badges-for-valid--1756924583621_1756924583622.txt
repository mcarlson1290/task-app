# Replit Instructions: Hide CUSTOM Frequency Badge

## Goal: Only show frequency badges for valid frequencies (WEEKLY, BI-WEEKLY, MONTHLY, QUARTERLY)

## Step 1: Find the Frequency Badge Code

Look for where the frequency badge shows "CUSTOM". It should look like:

```javascript
{task.recurringTaskId && (
  <span className={`...`}>
    {task.frequency === 'daily' ? 'WEEKLY' :
     task.frequency === 'biweekly' ? 'BI-WEEKLY' :
     task.frequency === 'monthly' ? 'MONTHLY' :
     task.frequency === 'quarterly' ? 'QUARTERLY' : 
     'CUSTOM'}  // ‚Üê This is the problem
  </span>
)}
```

## Step 2: Replace with Conditional Rendering

Replace the frequency badge code with this version that only shows for valid frequencies:

```javascript
{/* Frequency Type Badge - Only show for valid frequencies */}
{task.recurringTaskId && task.frequency && ['daily', 'biweekly', 'monthly', 'quarterly'].includes(task.frequency) && (
  <span className={`px-2 py-1 rounded text-xs font-bold uppercase text-white
    ${task.frequency === 'daily' ? 'bg-green-600' : 
      task.frequency === 'biweekly' ? 'bg-purple-600' : 
      task.frequency === 'monthly' ? 'bg-orange-600' : 
      task.frequency === 'quarterly' ? 'bg-pink-600' : 
      ''}
  `}>
    {task.frequency === 'daily' ? 'WEEKLY' :
     task.frequency === 'biweekly' ? 'BI-WEEKLY' :
     task.frequency === 'monthly' ? 'MONTHLY' :
     'QUARTERLY'}
  </span>
)}
```

## Step 3: Alternative Cleaner Version

For better readability, you can use this approach:

```javascript
{/* Frequency Type Badge */}
{(() => {
  if (!task.recurringTaskId || !task.frequency) return null;
  
  const frequencyMap = {
    'daily': { label: 'WEEKLY', color: 'bg-green-600' },
    'biweekly': { label: 'BI-WEEKLY', color: 'bg-purple-600' },
    'monthly': { label: 'MONTHLY', color: 'bg-orange-600' },
    'quarterly': { label: 'QUARTERLY', color: 'bg-pink-600' }
  };
  
  const freq = frequencyMap[task.frequency];
  if (!freq) return null; // Don't show badge if frequency not recognized
  
  return (
    <span className={`px-2 py-1 rounded text-xs font-bold uppercase text-white ${freq.color}`}>
      {freq.label}
    </span>
  );
})()}
```

## Step 4: Complete Badge Section

Here's the full badge section with the fix:

```javascript
<div className="flex flex-wrap items-center gap-1 mb-2">
  {/* RECURRING Badge - Always show for recurring tasks */}
  {task.recurringTaskId && (
    <span className="px-2 py-1 rounded text-xs font-bold uppercase bg-blue-600 text-white">
      RECURRING
    </span>
  )}
  
  {/* Frequency Badge - Only show for known frequencies */}
  {task.recurringTaskId && task.frequency && ['daily', 'biweekly', 'monthly', 'quarterly'].includes(task.frequency) && (
    <span className={`px-2 py-1 rounded text-xs font-bold uppercase text-white
      ${task.frequency === 'daily' ? 'bg-green-600' : 
        task.frequency === 'biweekly' ? 'bg-purple-600' : 
        task.frequency === 'monthly' ? 'bg-orange-600' : 
        'bg-pink-600'}
    `}>
      {task.frequency === 'daily' ? 'WEEKLY' :
       task.frequency === 'biweekly' ? 'BI-WEEKLY' :
       task.frequency === 'monthly' ? 'MONTHLY' :
       'QUARTERLY'}
    </span>
  )}
  
  {/* Day Badge */}
  <span className="px-2 py-1 rounded text-xs font-bold uppercase bg-teal-500 text-white">
    {getDayAbbreviation(task.dueDate)}
  </span>
  
  {/* Priority Badge */}
  {task.priority && (
    <span className={`px-2 py-1 rounded text-xs font-bold uppercase text-white
      ${task.priority === 'high' ? 'bg-red-500' : 
        task.priority === 'medium' ? 'bg-yellow-600' : 
        'bg-green-500'}
    `}>
      {task.priority}
    </span>
  )}
</div>
```

## Key Changes

1. Added condition: `['daily', 'biweekly', 'monthly', 'quarterly'].includes(task.frequency)`
2. This ensures the frequency badge only shows for recognized frequencies
3. Tasks without a valid frequency will only show the blue RECURRING badge
4. No more "CUSTOM" badges appearing

## Expected Result

Tasks with unknown/missing frequency:
- `RECURRING` `WED` `MEDIUM`

Tasks with valid frequency:
- `RECURRING` `WEEKLY` `WED` `MEDIUM`
- `RECURRING` `BI-WEEKLY` `WED` `MEDIUM`
- etc.
