## Step 1: Add Day of Week Display

In the task card, add a badge showing the day of week or task frequency:

```javascript
// In TaskCard component, add this function:
const getTaskDayDisplay = (task) => {
  if (task.frequency === 'monthly' || task.recurringPattern === 'monthly') {
    return { text: 'MONTHLY', color: '#7c3aed' }; // Purple
  }
  if (task.frequency === 'bi-weekly' || task.recurringPattern === 'bi-weekly') {
    return { text: 'BI-WEEKLY', color: '#0891b2' }; // Cyan
  }
  
  // For regular weekly tasks, show the day
  const dueDate = new Date(task.completionDate || task.dueDate);
  const dayName = dueDate.toLocaleDateString('en-US', { weekday: 'short' }).toUpperCase();
  return { text: dayName, color: '#059669' }; // Green
};

// In the task card header, add:
const dayDisplay = getTaskDayDisplay(task);
```
