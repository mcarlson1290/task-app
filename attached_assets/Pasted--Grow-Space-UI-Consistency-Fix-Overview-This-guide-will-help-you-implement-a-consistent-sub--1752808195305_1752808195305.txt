# Grow Space UI Consistency Fix

## Overview
This guide will help you implement a consistent sub-header navigation/filter bar across all pages in your Grow Space application.

## 1. Create a Shared Sub-Header Component

First, create a new component file that will be used across all pages:

**File: `components/SubHeader.jsx`**

```jsx
import React from 'react';

const SubHeader = ({ children, className = '' }) => {
  return (
    <div className={`sub-header ${className}`}>
      <div className="sub-header-content">
        {children}
      </div>
    </div>
  );
};

export default SubHeader;
```

## 4. Implementation Checklist

1. [ ] Create the `SubHeader.jsx` component
2. [ ] Add the CSS styles to your stylesheet
3. [ ] Import the SubHeader component in each page
4. [ ] Replace existing navigation/filter sections with the SubHeader
5. [ ] Test each page to ensure consistent spacing and styling
6. [ ] Adjust the main content area padding to account for the sub-header

## 5. Additional Notes

- The sub-header maintains a consistent height of 56px minimum
- All buttons use the same padding and font sizes
- The filter dropdown width issue is fixed with `min-width` property
- Responsive design is handled with `flex-wrap` for smaller screens
- The tab navigation style matches modern UI patterns

## 6. Quick Customization Guide

If you need to adjust any styling:

- **Button colors**: Modify the `.btn-primary` background-color
- **Spacing**: Adjust the `gap` values in `.sub-header-content` and `.filter-group`
- **Height**: Change the `min-height` in `.sub-header`
- **Font sizes**: Update the `font-size` values in button and input classes

This solution ensures that all pages have consistent spacing, button styling, and layout structure while maintaining flexibility for different content needs.
