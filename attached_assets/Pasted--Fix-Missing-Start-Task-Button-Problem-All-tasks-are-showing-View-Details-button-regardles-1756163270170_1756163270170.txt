# Fix Missing Start Task Button

## Problem
All tasks are showing "View Details" button regardless of their status. Tasks that haven't been started should show "Start Task" button instead.

## Solution for Replit

### Find the Task Card Component
Look for where task cards are rendered (likely `TaskCard.tsx` or within the Tasks page) and fix the button logic:

```typescript
// The button rendering logic should look something like this:
const renderTaskButton = (task) => {
  // Check task status to determine which button to show
  switch (task.status) {
    case 'pending':
    case 'not-started':
    case undefined: // In case status is missing
      return (
        <button 
          onClick={() => handleStartTask(task.id)}
          className="start-button"
          style={{
            backgroundColor: '#2D8028',
            color: 'white',
            padding: '8px 16px',
            border: 'none',
            borderRadius: '4px',
            cursor: 'pointer'
          }}
        >
          Start Task
        </button>
      );
      
    case 'in-progress':
    case 'started':
      return (
        <>
          <button 
            onClick={() => handlePauseTask(task.id)}
            className="pause-button"
          >
            Pause Task
          </button>
          <button 
            onClick={() => handleViewDetails(task.id)}
            className="details-button"
          >
            View Details
          </button>
        </>
      );
      
    case 'paused':
      return (
        <>
          <button 
            onClick={() => handleResumeTask(task.id)}
            className="resume-button"
          >
            Resume Task
          </button>
          <button 
            onClick={() => handleViewDetails(task.id)}
            className="details-button"
          >
            View Details
          </button>
        </>
      );
      
    case 'completed':
    case 'skipped':
      return (
        <button 
          onClick={() => handleViewDetails(task.id)}
          className="details-button"
        >
          View Details
        </button>
      );
      
    default:
      // Default to Start Task for safety
      return (
        <button 
          onClick={() => handleStartTask(task.id)}
          className="start-button"
        >
          Start Task
        </button>
      );
  }
};
```

  return <button>View Details</button>;

// RIGHT - Check the actual status
